# VHDL – Digital Circuits

This repository contains my work in VHDL completed as part of the Digital Circuits module during my studies at HAW Hamburg.

## Description

The code in this repository represents my introduction to hardware description and digital system design using VHDL. It includes implementations of basic digital components and exercises focused on understanding how hardware logic is modeled and simulated.

## Topics Covered

* Basic logic gates
* Combinational circuits
* Sequential circuits
* Digital system design concepts
* VHDL syntax and structure

## Structure

The files are organized based on exercises and concepts covered throughout the course.

## Note

This repository reflects my early experience with VHDL and digital design, forming a foundation for further work in hardware and embedded systems.
